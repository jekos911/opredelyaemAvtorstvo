# Лабораторные работы за семестр


В проекте находится несколько лаб. Ниже перечислены классы, являющиеся точками входа в программу

  
- AuthorIdentifier - определение авторства текста
  
- IKnowWhatYouSay - шифрование бита информации в тексте
  
- LanguageDefiner - определение типа грамматики
  
- Replace - класс, производящий замену символов в тексте



### AuthorIdentifier

Методом приближения Хи-квадрат высчитывается расстояние между матрицами-эталонами и анонимным текстом. Для расчетов используется статистика по общему числу биграмм и триграмм в текстах. Объем выборки определяется пользователем с клавиатуры



### IKnowWhatYouSay
Шифрование бита информации в введенном пользователем сообщении. При наличии символа Alt + 255 (пробел, имеющий иной код) в введенной строке программа выдаст ответ "DA", в противном случае - "NET"
